How It Works

This portal loads M3U playlists from any public URL and streams videos directly from your Jellyfin server using direct HTTP streaming.

Quick Tips

  • Make sure your M3U file is publicly accessible
  • URLs should be direct streaming links
  • Use VLC for best compatibility
  • Check console for loading errors

Troubleshooting

  • If videos don't play, check URL format
  • Ensure Jellyfin server is accessible
  • Try different browser or VLC player
  • Clear browser cache if needed
